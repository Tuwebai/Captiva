## Diferencia clave: objetivo único vs navegación abierta

Una landing está diseñada para una conversión concreta. Un sitio web tradicional reparte la atención entre secciones y suele funcionar mejor para branding amplio.

## Cuándo elegir landing page

- Necesitás generar consultas rápidas.
- Tenés una oferta específica.
- Querés medir campañas por fuente.

## Cuándo elegir sitio web completo

- Tenés múltiples líneas de negocio.
- Necesitás documentación extensa.
- El objetivo principal es institucional.

## Recomendación para negocios en crecimiento

La estrategia más eficiente suele ser comenzar con una landing de alto enfoque y escalar luego a sitio completo. En Captiva podés validar esa base con la biblioteca de demos en [/captiva/demos](/captiva/demos).
